REACT介绍

学习任何一项新知识、新技能，首先要了解的就是三个W，也就是what、why、how————无数人的经验

  • WHAT: React是Facebook内部开源出来的一个前端ui框架，内容本身并不很复杂，前期上手学习的难点主要在于开发环境和react技术体系的搭建与理解
  • WHY: 现在前端世界可以说是三分天下，它和vue与angula不同在哪里呢？
    • react 数据单向传输，使用flux架构，不同于后两者使用的mvvm
    • react 使用jsx写法，可以理解为js包含html 其他两个可对应理解为html包含js(个人感觉，不一定对哈)
    • 也稍微接触过一点点vue和angular，前期上手难度vue < react < angular
    • 为什么选择学习react呢？主要是近期可能会使用它
  • HOW: 那怎么开始上手呢？我的学习习惯是先来个Hello World!(相信很多人都是哈)
